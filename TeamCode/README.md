## ü§ñ **ROBOT SYSTEMS**

### **Drive System**
- 4 mecanum wheels (omnidirectional movement)
- 2 drive modes: Robot-relative and Field-relative
- 3 speed modes: Slow (50%), Normal (75%), Turbo (100%)

### **Intake System**
- Front intake (IF) + Back intake (IB)
- Collects and stores game elements
- Bidirectional (collect and eject)

### **Shooter System**
- Dual motors (SL + SR) for consistency
- Launches stored game elements
- Adjustable speed via FTC Dashboard
