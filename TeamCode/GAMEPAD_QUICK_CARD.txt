# ğŸ® FIELD-RELATIVE MODE DIFFERENCES

When using **"Penguinauts: Field Relative Drive"**:

### ADDITIONAL CONTROLS:

**A Button** = Reset Field Orientation
- Press when robot is facing forward
- Sets current direction as "field forward"

**Left Bumper (LB)** = Robot-Relative Mode (hold)
- Hold LB to drive like RC car
- Release LB to return to field-relative

**Right Bumper (RB)** = Slow Mode (50%)
- Same as standard mode

### HOW IT WORKS:
- **Left stick** direction = direction on FIELD
- Robot moves that direction regardless of rotation
- Great for competition!
- Takes practice to master
