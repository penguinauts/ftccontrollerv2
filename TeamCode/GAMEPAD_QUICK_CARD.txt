**Shooter Tips:**
- Press RT only when ready to shoot
- Keep RT pressed until ball leaves shooter
- Press B immediately after shooting

## ğŸ¯ SHOOTER OPERATION

### Normal Shooting Sequence:

1. **Position Robot** - Drive to shooting position
2. **Prepare** - Ensure ball is in shooter mechanism
3. **SHOOT** - Press & hold Right Trigger
4. **Stop** - Press B button after ball shoots
5. **Repeat** - Reload and shoot again

### Speed Adjustment (via FTC Dashboard):

The shooter speed can be adjusted in real-time:

1. Connect laptop to robot WiFi
2. Open browser: `http://192.168.43.1:8080/dash`
3. Find **SHOOTER_POWER** variable
4. Adjust slider (0.0 to 1.0)
5. Default: **1.0** (100% speed)
6. Test different speeds for accuracy

**Common Settings:**
- **1.0** = Maximum power (long distance)
- **0.85** = Good balance (medium distance)
- **0.70** = Close range shots
- **0.50** = Very close/testing

# ğŸ® FIELD-RELATIVE MODE DIFFERENCES

When using **"Penguinauts: Field Relative Drive"**:

### ADDITIONAL CONTROLS:

**A Button** = Reset Field Orientation
- Press when robot is facing forward
- Sets current direction as "field forward"

**Left Bumper (LB)** = Robot-Relative Mode (hold)
- Hold LB to drive like RC car
- Release LB to return to field-relative

**Right Bumper (RB)** = Slow Mode (50%)
- Same as standard mode

### HOW IT WORKS:
- **Left stick** direction = direction on FIELD
- Robot moves that direction regardless of rotation
- Great for competition!
- Takes practice to master
